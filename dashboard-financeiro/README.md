# 📊 Dashboard Financeiro

## 📌 Sobre o Projeto

Dashboard desenvolvido em Power BI com o objetivo de fornecer uma visão consolidada dos principais indicadores financeiros, permitindo acompanhar receitas, custos, despesas, lucro e fluxo de caixa de forma rápida e intuitiva.

---

## 🎯 Objetivos

* Monitorar a saúde financeira do negócio.
* Acompanhar a evolução das receitas e despesas.
* Analisar o fluxo de caixa ao longo do tempo.
* Facilitar a tomada de decisões baseada em dados.

---

## 🛠️ Tecnologias Utilizadas

* Power BI
* Power Query
* DAX
* Modelagem de Dados

---

## 📈 Indicadores Monitorados

* Receita
* Custos
* Despesas
* Lucro
* Fluxo de Caixa
* Receita por Cliente
* Receita por Tipo de Conta
* Pagamentos por Tipo

---

## 🚀 Competências Demonstradas

* Transformação e tratamento de dados com Power Query
* Criação de medidas e KPIs utilizando DAX
* Modelagem de dados para análise financeira
* Desenvolvimento de dashboards interativos
* Storytelling com dados e visualização de informações
